# Twitter Trend & Sentiment Analysis Dashboard

This project is a **Twitter Trend & Sentiment Analysis Dashboard** built with Python, Dash, and Plotly. It analyzes Twitter data to extract trending keywords, sentiment distribution, daily sentiment trends, and displays an interactive word cloud. Users can filter tweets by sentiment using a dropdown.

---

## Features

- **Sentiment Analysis:** Classifies tweets as Positive, Negative, or Neutral using TextBlob.
- **Trending Keywords:** Displays the top 10 most frequent keywords in the tweets.
- **Daily Sentiment Trends:** Line chart showing sentiment counts over time.
- **Word Cloud:** Visual representation of the most common words.
- **Filter Tweets:** Interactive dropdown to filter and display tweets by sentiment.

---

## Technologies Used

- Python 3.x
- Pandas
- TextBlob (for sentiment analysis)
- Scikit-learn (CountVectorizer for keyword extraction)
- Dash & Dash Bootstrap Components (for interactive web dashboard)
- Plotly Express (for visualizations)
- WordCloud (for word cloud generation)
- pyngrok (to expose local Dash app to public URL)
